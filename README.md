# Крипто-реликвия дня

Раз в день весь мир открывает один кейс, запечатанный хешем сегодняшнего биткоин-блока.
Что выпадет — зависит от твоего ника. Реролла нет. Приходи завтра за новым.

Одно ядро (`web/engine.mjs`), две «морды»: **веб-сайт** и **Telegram-бот**.

## Почему это работает без базы и бесплатно

Дроп **детерминирован**: `дата + хеш блока + ник → предмет`. Одинаковый ввод даёт
одинаковый результат у всех и всегда, поэтому **хранить в базе нечего** — всё
вычисляется на лету. Это же делает проект provably-fair: исходник открыт, подкрутить нельзя.

## Веб: запуск локально

Нужен любой статический сервер (из-за ES-модулей нельзя открывать файл напрямую):

```bash
cd web
npx serve .            # или: python -m http.server 3000
```

Открой `http://localhost:3000`.

### Деплой (бесплатно, навсегда)
- **GitHub Pages**: залей содержимое `web/` в репозиторий, включи Pages.
- **Cloudflare Pages** / **Netlify** / **Vercel**: укажи `web/` как корень, build-команды не нужны.

### Виральные крючки, уже встроенные
- Кнопка **«Поделиться в 𝕏»** — открывает готовый твит через intent-ссылку (без API, без ключей).
- **Deep-link** `?u=ник` — по ссылке открывается превью чужого дропа с призывом «вбей свой». Это и есть цикл «проверь свой».

## Telegram-бот: запуск

```bash
cd bot
npm install
BOT_TOKEN=<токен от @BotFather> npm start
```

Команды: `/open`, `/collection`, `/streak`, `/odds`.
В Telegram сценарий «напиши боту → получи дроп» **бесплатен и без лимитов на чтение** (в отличие от X API).

### Деплой бота бесплатно
- Простейше: **Railway** / **Render** free-инстанс (polling, как сейчас).
- Правильно и «спит бесплатно»: **Cloudflare Workers** через webhook, состояние в **Workers KV** вместо `store.json`.

## Известные упрощения MVP (честно)

- **Маяк дня** — берётся текущий tip-блок и кэшируется на дату у каждого клиента отдельно.
  Между клиентами он может отличаться на пару блоков в течение дня. Чистый фикс:
  фиксировать блок, ближайший к 00:00 UTC, единый для всех.
- **Веб-коллекция** живёт в `localStorage` (легко «читерить», нет глобального лидерборда).
  Для честного лидерборда добавь бесплатный serverless-KV.
- **Токеномики/вывода ценностей нет** — это игрушка. Так и надо для старта: без реальных
  денег внутри это не подпадает под законы об азартных играх.

## Как расширять

- OG-картинка под каждый дроп → один serverless-эндпоинт (Cloudflare/Vercel), рисующий
  изображение по `?u=ник`. Тогда в ленте X разворачивается карточка предмета.
- Общий лидерборд редких дропов, «музей» мификов, обмен между игроками.
